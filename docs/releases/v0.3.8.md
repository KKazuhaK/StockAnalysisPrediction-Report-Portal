## Upgrade notes

v0.3.8 has no database schema changes and can be upgraded directly from v0.3.7.

## Changes

- Fix Mermaid flowchart labels disappearing after SVG sanitization. Charts now use native SVG text while retaining strict Mermaid security and the existing sanitizer policy.
- Add an interactive chart viewport to report reading and Chat: scroll or use a trackpad to zoom, drag to pan, and use the toolbar to zoom or reset the view.
- Limit tall chart viewports to the visible browser area so large flowcharts remain practical to inspect.
- Add localized chart controls for Simplified Chinese, Traditional Chinese, and English.

## Known limitations

- Markdown downloads continue to preserve the complete Mermaid source.
- PDF and daily ZIP exports still show Mermaid source blocks rather than rendered charts.
