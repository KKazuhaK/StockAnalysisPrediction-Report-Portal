## Upgrade notes

v0.3.10 has no database schema changes and can be upgraded directly from v0.3.9.

## Changes

- Match browser and PDF Mermaid cache keys for fenced code blocks indented by up to three spaces, following CommonMark fence indentation rules. Reports generated with indented Mermaid fences now embed the cached chart in PDF exports instead of showing the source fallback.
- Add frontend and backend regression coverage for indented Mermaid fences.

## Operational note

The Mermaid PDF cache is in memory. After upgrading or restarting the server, open or refresh a report once and wait for its charts to render before exporting its PDF.
